invokeScript

Invokes a script on the VM

Method definition
protected abstract void InvokeScript(string chain, byte[] script, int id, Action<byte[], string> callback);
Parameters
  1. string Chain that the script needs to run
  2. byte[] The script call in bytes.
  3. int The id.
  4. Action Call back that returns a result byte[] that contains all the information returned script, and a string.
Returns

Byte[] - A Byte array object with the return from the script, or error if something wents wrong.

  • result: string - Return of the method.
