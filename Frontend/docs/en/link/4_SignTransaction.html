SignTx ( Signed Transaction )

Returns the account name and balance of given address.

Method definition
protected abstract void SignTransaction(string platform, SignatureKind kind, string chain, byte[] script, byte[] payload, int id, Action<Hash, string> callback);
Parameters
  1. string Address of account.
  2. string Signature Kind.
  3. string Chain.
  4. Action Call back that returns an Account object and a string.
Returns

Hash - A transaction hash, or error if transaction went wrong.

  • result: string - Hash address.
