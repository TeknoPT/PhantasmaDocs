WriteArchive

Returns the account name and balance of given address.

Method definition
protected abstract void WriteArchive(Hash hash, int blockIndex, byte[] data, Action<bool, string> callback);
Parameters
  1. Hash Address of account.
  2. Action Call back that returns an Account object and a string.
Returns

Account - A Account object, or error if address is invalid.

  • address: string - Given address.
  • name: string - Name of given address.
  • stakes: Array - Array of stake objects.
