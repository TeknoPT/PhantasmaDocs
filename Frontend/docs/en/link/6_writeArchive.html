WriteArchive

Writes to an Archive.

Method definition
protected abstract void WriteArchive(Hash hash, int blockIndex, byte[] data, Action<bool, string> callback);
Parameters
  1. Hash Hash of the block.
  2. int Block index .
  3. byte[] Data to write to archive
  4. Action Call back that returns a boolean (true if it was accepted) and a string.
Returns

Archive - A Archive object, or error if address is invalid.

  • address: string - Given address.
